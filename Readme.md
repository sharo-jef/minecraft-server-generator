# Minecraft Server Generator
Generator for Minecraft Vanilla Server.

![](https://i.gyazo.com/8e107fcfe4f7d44ea6178e18287ed2b0.png)

## Requirements
- Node.js (^v13)

## Usage
### Generate
```bash
npm i
npm run build
```

### Boot
```bash
npm start
```
or
```bash
# Windows (cmd)
boot
# Windows (pwsh)
./boot
# Other (bash)
./boot.sh
```
